## Install MYSQL on Linux :
```bash
$ sudo apt install mariadb-server mariadb
```
```bash
$ mariadb-service-convert
```
```bash
$ sudo systemctl start mariadb
```
```bash
$ mysql -u root
```

## Install mysql.connector :
```bash
$ pip install mysql-connector-python
```
